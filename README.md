# SearchRewards Bot

Telegram bot for fast searches that rewards users for referrals in USD cents, logs activity, and lets users request manual withdrawals while admins manage payouts and fraud flags.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
